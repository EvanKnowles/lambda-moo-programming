Installation:

First, copy the extension-goblin.c file into a directory with the freshly
  untarred server source.  Add extension-goblin.c to the CSRCS line of
  Makefile.in, add
          extern void register_goblin(void);
  to bf_register.h and add
         register_goblin
  to the list of functions in functions.c.

Also, you must remove the bf_min and bf_max functions from numbers.c
Don't forget to remove the register_function lines at the end of the 
file as well...
